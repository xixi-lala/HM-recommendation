# 46维特征消融报告

## 实验方法

- **框架**：step5_fixed（前5天训练 / 后2天留出 / 早停50轮）
- **方式**：从全46维出发，每次剔除1个特征维度，剩余45维训练并评估 Holdout MAP@12
- **判定**：Δ < -0.0001 → 有效；Δ ≥ -0.0001 → 噪声
- **基线**：全46维 Holdout=0.02539, Train=0.04974, 过拟合=0.02435, 轮数=170

## 三、关键发现

### 1. 最重要的3个特征
- **F_count**（客户购买频次）：去掉后 Holdout 下降 0.00167
- **cf_score**（协同过滤得分）：去掉后下降 0.00163
- **n_buyers**（商品被购买人数）：去掉后下降 0.00161

### 2. 最有害的3个特征（去掉后反而提升）
- **club_member_status_le**：去掉后 Holdout **上升 0.00129**，过拟合从 0.02435 升至 0.02546（但轮数涨到227）
- **text_emb_10**：去掉后上升 0.00085
- **M_spend**：去掉后上升 0.00084

### 3. 文本嵌入的尴尬
- 20维文本嵌入中，7维（35%）是纯噪声
- 这些维度来自 TF-IDF → SVD，本身不具备语义可解释性
- 噪声维度去掉后推理速度提升、过拟合降低

### 4. 客户特征的问题
- 8维客户特征中3维是噪声：club_member_status_le、M_spend、avg_price_user
- 这些特征可能是训练集统计不稳定或信息已被其他特征覆盖

### 5. 预期效果
- 全46维 → 34维，特征数减少 26%
- 去掉12个噪声特征后，预计过拟合降低，模型更稳定
- 训练/推理速度提升约 26%
